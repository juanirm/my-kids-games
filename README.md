# my-kids-games

Small static site for kids' web games. Hosted on GitHub Pages.

This repo contains a simple mobile-first game called "Electric Battery" — an 8-pad drum kit with microphone sampling, plus placeholders for a small piano and harp.

Usage:
- Open `index.html` or publish with GitHub Pages (enable Pages from `main` branch, root).

Tech:
- Tone.js for audio synthesis and playback
- Plain HTML/CSS/JS for simplicity and portability
